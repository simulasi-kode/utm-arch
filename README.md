# UTM ARCH
Workflow installation Arch Linux using [UTM Virtual Machine for Mac](https://mac.getutm.app/). In this case i'm using Macbook pro 13-inch 2017 Operation system Ventura 13.7.8. Please download here [ISO file](https://fastly.mirror.pkgbuild.com/iso/2025.11.01/archlinux-x86_64.iso) if you don't have a file.

**UTM Setup and guide**
- Create virtual manager with UTM using EFI
- choose linux, and just start the Machine.

## [Installation](Installation/README.md)
This is a basic installation using terminal from [Wiki Arch Linux](https://wiki.archlinux.org/title/Installation_guide) and Watch [Youtube](https://www.youtube.com/watch?v=cOobSmI-XgA)

## [Post-Install](Post-Install/Config.md)
Login back to Virtual Machine to finish configuration, make sure everything is update run this command ```pacman -Syu```.
